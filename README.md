# Installation/Distribution

Released builds are available from:

    http://dl.bintray.com/javanarior/maven

Add the dependency:

    <groupId>de.javanarior.library</groupId>
    <artifactId>valueobject-library</artifactId>
    <version>0.1.0</version>
